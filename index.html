Keyboard shortcuts

  • Enter run the search from any criteria field
  • + / - zoom the map in / out
  • G go to coordinates
  • R toggle the ruler tool
  • ? open this help
  • Esc close the active dialog, tool or popup

Map tools

  • 📏 ruler — measure a distance in blocks (two clicks)
  • 📍 marker — pin a named point on the map
  • ⬚ selection — select an area to export a PNG or copy its coordinates
  • ▧ zone — draw a named colored zone, kept in your browser
  • 🌀 portal — plan a linked Nether portal (÷8 / ×8 with the game's rounding)
